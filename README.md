## 🎯 Tentang Project

**SANS Hub** adalah platform berbasis web yang menyediakan kalkulator dan tools pendukung keputusan klinis untuk tenaga medis, khususnya di lingkungan ICU dan perawatan pediatri.

Dikembangkan oleh tim Keperawatan Rumah Sakit Adhyaksa Jawa Timur sebagai alat bantu yang cepat, akurat, dan mudah diakses.
